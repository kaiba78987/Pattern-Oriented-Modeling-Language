This project is only an incomplete prototype for research purposes.

You can build and run the project using Gradle.

If you don't have Gradle installed, you can use the Gradle wraper via gradlew or gradlew.bat files.

Most important commands:
gradle build: Builds the project.
gradle test: Tests the application.
gradle scaladoc: Generates the documentation.
gradle run: Runs the application with presetted input and output (check args in build.gradle)
